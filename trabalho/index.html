O que é HTTP?

Protocolo que permite a troca de informações entre navegador e servidor. Não é seguro.

O que é HTTPS?

Versão segura do HTTP, com criptografia para proteger os dados.

O que é Cliente?

Dispositivo ou programa (como navegador) que solicita informações ao servidor.

O que é Servidor?

Computador que armazena dados e responde às solicitações dos clientes.
